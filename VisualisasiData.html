Eksplorasi Data Iris dengan MySQL, PostgreSQL, dan Power BI

Mengumpulkan Data Awal#

Dataset Iris adalah dataset klasik dalam analisis data dan pembelajaran mesin. Dataset ini terdiri dari 150 baris dengan 4 atribut numerik:

  • Sepal Length
  • Sepal Width
  • Petal Length
  • Petal Width

Serta 1 atribut kategori yaitu species (Setosa, Versicolor, Virginica).

Tujuan Proyek

  1. Membuat database Iris di MySQL dan PostgreSQL.
  2. Menghubungkan database tersebut dengan Power BI.
  3. Melakukan analisis deskriptif berupa min, max, dan rata-rata setiap atribut.
  4. Menampilkan distribusi jumlah data setiap kelas dalam bentuk grafik batang.

Database#

Struktur Database

```sql CREATE DATABASE iris_db;

USE iris_db;

CREATE TABLE iris ( id INT AUTO_INCREMENT PRIMARY KEY, sepal_length FLOAT, sepal_width FLOAT, petal_length FLOAT, petal_width FLOAT, species VARCHAR(20) );

Proses connect Postgresql di DBeaver

  1. Buka DBeaver → New Database Connection → pilih MySQL/Postgrasql.
  2. Masukkan hostname, port, username, dan password(mysql dan postgresql).postgresql juga bisa menggunakan url seperti yang sudah dilampirkan pada informasi koneksi diatas

Integrasi Database ke Power BI#

  1. Buka Power BI Desktop.
  2. Pilih Get Data → MySQL Database atau PostgreSQL Database.
  3. Masukkan hostname, port, username, dan password sesuai konfigurasi.
  4. Pilih database iris_db dan tabel iris.
  5. Data siap divisualisasikan.

Deskripsi Data

Dataset Iris adalah dataset klasik dalam analisis data dan pembelajaran mesin. Dataset ini terdiri dari 150 baris dengan 4 atribut numerik:

  • Sepal Length
  • Sepal Width
  • Petal Length
  • Petal Width
Kolom Tipe Data Keterangan
sepal_length FLOAT/REAL Panjang sepal dalam cm
sepal_width FLOAT/REAL Lebar sepal dalam cm
petal_length FLOAT/REAL Panjang petal dalam cm
petal_width FLOAT/REAL Lebar petal dalam cm
species VARCHAR Jenis bunga iris

Interpretasi Warna Batang

  • 🟦 Biru muda → Sum of petal_length (total panjang mahkota)

  • 🔵 Biru tua → Sum of petal_width (total lebar mahkota)

  • 🟧 Oranye → Sum of sepal_length (total panjang kelopak)

  • 🟪 Ungu → Sum of sepal_width (total lebar kelopak)

    Tabel Ringkasan Total/Kolom

Species Sum of petal_length Sum of petal_width Sum of sepal_length Sum of sepal_width
Iris-virginica 277.60 101.30 329.40 148.70
Iris-versicolor 213.00 66.30 296.80 138.50
Iris-setosa 73.20 12.20 250.30 170.90

Interpretasi Tabel

  1. Iris-virginica
    • Total petal_length dan sepal_length paling tinggi.
    • Menunjukkan bahwa spesies ini memiliki kelopak dan mahkota yang relatif panjang.
  2. Iris-versicolor
    • Semua nilai total berada pada posisi menengah di antara dua spesies lainnya.
  3. Iris-setosa
    • Memiliki total petal_length dan petal_width paling rendah.
    • Menunjukkan mahkota paling kecil, tetapi sepal_width relatif lebih besar dibandingkan spesies lain.

Hubungan dengan Grafik Batang

  • Batang biru muda (petal_length) paling tinggi untuk Virginica → sesuai dengan nilai 277.60.
  • Batang biru tua (petal_width) paling rendah untuk Setosa → sesuai dengan nilai 12.20.
  • Batang ungu (sepal_width) tertinggi justru pada Setosa (170.90), berbeda dengan pola fitur lainnya.
